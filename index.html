Skillek

HTML

HTML (HyperText Markup Language) a weboldalak alapját képezi, amely strukturálja a tartalmat, és meghatározza az elemek, mint például címek, bekezdések, képek és linkek elrendezését.

CSS

CSS (Cascading Style Sheets) a weboldalak megjelenésének kialakítására szolgál, lehetővé téve a színek, betűtípusok, margók és egyéb stílusok meghatározását, hogy vizuálisan vonzóvá tegyék a tartalmat.

Bootstrap

A Bootstrap egy nyílt forráskódú CSS keretrendszer, amely gyors és reszponzív webfejlesztést tesz lehetővé, előre elkészített stílusokkal, komponensekkel és JavaScript pluginekkel.

JavaScript

A JavaScript egy dinamikus programozási nyelv, amely interaktív funkciók hozzáadását teszi lehetővé a weboldalakhoz, mint például animációk, űrlapellenőrzés és dinamikus tartalom frissítése.

TypeScript

A TypeScript egy nyílt forráskódú programozási nyelv, amely a JavaScript szuperszettje, és amely típusbiztonságot ad a kódhoz, segítve a fejlesztőket a hibák korai észlelésében és az alkalmazások karbantarthatóságának javításában.

Tudj meg többet
Angular

Az Angular egy TypeScript alapú keretrendszer, amelyet a Google fejlesztett, és amely lehetővé teszi összetett, dinamikus és reszponzív egyoldalas alkalmazások (SPA) létrehozását.

Tudj meg többet
Node JS

A Node.js egy nyílt forráskódú, szerveroldali futtatási környezet, amely lehetővé teszi JavaScript használatát szerveroldali programozáshoz, támogatva a skálázható és nagy teljesítményű alkalmazások fejlesztését.

MongoDB

A MongoDB egy dokumentum-orientált NoSQL adatbázis, amely nagy mennyiségű adat tárolására és kezelésére alkalmas, rugalmas, skálázható és könnyen használható JSON-szerű dokumentumokkal dolgozva.
